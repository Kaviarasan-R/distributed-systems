# Azure Service Bus

Azure Service Bus is a fully managed enterprise message broker with message queues and publish-subscribe topics.

## [Advanced Features](https://learn.microsoft.com/en-gb/azure/service-bus-messaging/advanced-features-overview)

- Topics and Subscriptions
- Message sessions
- Auto-forwarding
- Transactions
- Filters and actions
- Duplicate detection
